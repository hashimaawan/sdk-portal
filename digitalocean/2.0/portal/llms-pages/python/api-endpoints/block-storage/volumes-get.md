# Volumes Get

Source: https://raw.githubusercontent.com/hashimaawan/sdk-portal/main/digitalocean/2.0/portal/#/python/x-redirect/JTI0ZSUyRkJsb2NrJTI1MjBTdG9yYWdlJTJGdm9sdW1lc19nZXQ

To show information about a block storage volume, send a GET request to `/v2/volumes/$VOLUME_ID`.

```python
def volumes_get(self,
               volume_id)
```


# Authentication

This endpoint requires [bearer_auth](https://raw.githubusercontent.com/hashimaawan/sdk-portal/main/digitalocean/2.0/portal/llms-pages/python/getting-started/quickstart/authorization.md)


# Parameters

| Parameter | Type | Tags | Description |
|  --- | --- | --- | --- |
| `volume_id` | `uuid\|str` | Template, Required | The ID of the block storage volume. |


# Response Type

**200**: The response will be a JSON object with a key called `volume`. The value will be an object containing the standard attributes associated with a volume.

This method returns an [`ApiResponse`](https://raw.githubusercontent.com/hashimaawan/sdk-portal/main/digitalocean/2.0/portal/llms-pages/python/sdk-infrastructure/utilities/apiresponse.md) instance. The `body` property of this instance returns the response data which is of type [`V2VolumesResponse1`](https://raw.githubusercontent.com/hashimaawan/sdk-portal/main/digitalocean/2.0/portal/llms-pages/python/models/structures/v2-volumes-response-1.md).


# Example Usage

```python
volume_id = '7724db7c-e098-11e5-b522-000f53304e51'

result = block_storage_api.volumes_get(volume_id)

if result.is_success():
    print(result.body)
elif result.is_error():
    print(result.errors)
```
