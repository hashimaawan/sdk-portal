# Invoices Get by UUID

Source: https://raw.githubusercontent.com/hashimaawan/sdk-portal/main/digitalocean/2.0/portal/#/python/x-redirect/JTI0ZSUyRkJpbGxpbmclMkZpbnZvaWNlc19nZXRfYnlVVUlE

To retrieve the invoice items for an invoice, send a GET request to `/v2/customers/my/invoices/$INVOICE_UUID`.

```python
def invoices_get_by_uuid(self,
                        invoice_uuid)
```


# Authentication

This endpoint requires [bearer_auth](https://raw.githubusercontent.com/hashimaawan/sdk-portal/main/digitalocean/2.0/portal/llms-pages/python/getting-started/quickstart/authorization.md)


# Parameters

| Parameter | Type | Tags | Description |
|  --- | --- | --- | --- |
| `invoice_uuid` | `str` | Template, Required | UUID of the invoice |


# Response Type

**200**: The response will be a JSON object with a key called `invoice_items`. This will be set to an array of invoice item objects.

This method returns an [`ApiResponse`](https://raw.githubusercontent.com/hashimaawan/sdk-portal/main/digitalocean/2.0/portal/llms-pages/python/sdk-infrastructure/utilities/apiresponse.md) instance. The `body` property of this instance returns the response data which is of type [`V2CustomersMyInvoicesResponse1`](https://raw.githubusercontent.com/hashimaawan/sdk-portal/main/digitalocean/2.0/portal/llms-pages/python/models/structures/v2-customers-my-invoices-response-1.md).


# Example Usage

```python
invoice_uuid = '22737513-0ea7-4206-8ceb-98a575af7681'

result = billing_api.invoices_get_by_uuid(invoice_uuid)

if result.is_success():
    print(result.body)
elif result.is_error():

```
