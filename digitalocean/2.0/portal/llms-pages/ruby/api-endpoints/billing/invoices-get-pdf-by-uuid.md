# Invoices Get Pdf by UUID

Source: https://raw.githubusercontent.com/hashimaawan/sdk-portal/main/digitalocean/2.0/portal/#/ruby/x-redirect/JTI0ZSUyRkJpbGxpbmclMkZpbnZvaWNlc19nZXRfcGRmQnlVVUlE

To retrieve a PDF for an invoice, send a GET request to `/v2/customers/my/invoices/$INVOICE_UUID/pdf`.

```ruby
def invoices_get_pdf_by_uuid(invoice_uuid)
```


# Authentication

This endpoint requires [bearer_auth](https://raw.githubusercontent.com/hashimaawan/sdk-portal/main/digitalocean/2.0/portal/llms-pages/ruby/getting-started/quickstart/authorization.md)


# Parameters

| Parameter | Type | Tags | Description |
|  --- | --- | --- | --- |
| `invoice_uuid` | `String` | Template, Required | UUID of the invoice |


# Response Type

**200**: The response will be a PDF file.

This method returns an [`ApiResponse`](https://raw.githubusercontent.com/hashimaawan/sdk-portal/main/digitalocean/2.0/portal/llms-pages/ruby/sdk-infrastructure/utilities/apiresponse.md) instance. The `data` property of this instance returns the response data which is of type `Mixed`.


# Example Usage

```ruby
invoice_uuid = '22737513-0ea7-4206-8ceb-98a575af7681'

result = billing_api.invoices_get_pdf_by_uuid(invoice_uuid)

if result.success?
  puts result.data
elsif result.error?
  warn result.errors
end
```
