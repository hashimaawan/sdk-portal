# Remove-Audiobooks-User

Source: https://raw.githubusercontent.com/hashimaawan/sdk-portal/main/spotify/portal/#/net-standard-library/x-redirect/JTI0ZSUyRkF1ZGlvYm9va3MlMkZyZW1vdmUtYXVkaW9ib29rcy11c2Vy

Remove one or more audiobooks from the Spotify user's library.

```csharp
RemoveAudiobooksUserAsync(
    string ids)
```


# Authentication

This endpoint requires [oauth_2_0](https://raw.githubusercontent.com/hashimaawan/sdk-portal/main/spotify/portal/llms-pages/net-standard-library/getting-started/quickstart/authorization.md)


# Parameters

| Parameter | Type | Tags | Description |
|  --- | --- | --- | --- |
| `ids` | `string` | Query, Required | - |


# Requires scope

## oauth_2_0

`user-library-modify`


# Response Type

**200**: Audiobook(s) have been removed from the library

`Task`
