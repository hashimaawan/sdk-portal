# Overview

Source: https://raw.githubusercontent.com/hashimaawan/sdk-portal/main/hetzner/1.0.0/portal/#/php/x-redirect/JTI0aCUyRl9fYXBpX3JlZmVyZW5jZSUyRlNlcnZlciUyNTIwVHlwZXMlMkZPdmVydmlldw

Server types define kinds of Servers offered. Each type has an hourly and a monthly cost. You will pay whichever cost is lower for your usage of this specific Server. Costs may differ between Locations.

Currency for all amounts is €. All prices exclude VAT.


# Get singleton instance

The singleton instance of the `ServerTypesApi` class can be accessed from the API Client.

```
$serverTypesApi = $client->getServerTypesApi();
```
