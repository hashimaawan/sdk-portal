# Overview

Source: https://raw.githubusercontent.com/hashimaawan/sdk-portal/main/hetzner/#/java/x-redirect/JTI0aCUyRl9fYXBpX3JlZmVyZW5jZSUyRlZvbHVtZXMlMkZPdmVydmlldw

A Volume is a highly-available, scalable, and SSD-based block storage for Servers.

Pricing for Volumes depends on the Volume size and Location, not the actual used storage.

Please see [Hetzner Docs](https://docs.hetzner.com/cloud/#Volumes) for more details about Volumes.


# Get instance

An instance of the `VolumesController` class can be accessed from the API Client.

```
VolumesController volumesController = client.getVolumesController();
```
