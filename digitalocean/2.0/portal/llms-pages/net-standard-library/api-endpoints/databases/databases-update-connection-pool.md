# Databases Update Connection Pool

Source: https://raw.githubusercontent.com/hashimaawan/sdk-portal/main/digitalocean/2.0/portal/#/net-standard-library/x-redirect/JTI0ZSUyRkRhdGFiYXNlcyUyRmRhdGFiYXNlc191cGRhdGVfY29ubmVjdGlvblBvb2w

To update a connection pool for a PostgreSQL database cluster, send a PUT request to  `/v2/databases/$DATABASE_ID/pools/$POOL_NAME`.

```csharp
DatabasesUpdateConnectionPoolAsync(
    Guid databaseClusterUuid,
    string poolName,
    Models.V2DatabasesPoolsRequest body)
```


# Authentication

This endpoint requires [bearer_auth](https://raw.githubusercontent.com/hashimaawan/sdk-portal/main/digitalocean/2.0/portal/llms-pages/net-standard-library/getting-started/quickstart/authorization.md)


# Parameters

| Parameter | Type | Tags | Description |
|  --- | --- | --- | --- |
| `databaseClusterUuid` | `Guid` | Template, Required | A unique identifier for a database cluster. |
| `poolName` | `string` | Template, Required | The name used to identify the connection pool. |
| `body` | [`V2DatabasesPoolsRequest`](https://raw.githubusercontent.com/hashimaawan/sdk-portal/main/digitalocean/2.0/portal/llms-pages/net-standard-library/models/structures/v2-databases-pools-request.md) | Body, Required | - |


# Response Type

**204**: The action was successful and the response body is empty.

`Task`


# Example Usage

```csharp
Guid databaseClusterUuid = new Guid("9cc10173-e9ea-4176-9dbc-a4cee4c4ff30");
string poolName = "backend-pool";
V2DatabasesPoolsRequest body = new V2DatabasesPoolsRequest
{
    Db = "defaultdb",
    Mode = "transaction",
    Size = 10,
    User = "doadmin",
};

try
{
    await databasesApi.DatabasesUpdateConnectionPoolAsync(
        databaseClusterUuid,
        poolName,
        body
```
