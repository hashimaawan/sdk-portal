features, and configuration details already set up for you. They can be found in the
[DigitalOcean Marketplace](https://www.digitalocean.com/docs/marketplace).


# Create Instance

The instance of the `M1ClickApplicationsApi` class can be created using the API Client.

```
const m1ClickApplicationsApi = new M1ClickApplicationsApi(client);
```
