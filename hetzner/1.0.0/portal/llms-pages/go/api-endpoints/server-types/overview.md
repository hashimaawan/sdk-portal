# Overview

Source: https://raw.githubusercontent.com/hashimaawan/sdk-portal/main/hetzner/1.0.0/portal/#/go/x-redirect/JTI0aCUyRl9fYXBpX3JlZmVyZW5jZSUyRlNlcnZlciUyNTIwVHlwZXMlMkZPdmVydmlldw

Server types define kinds of Servers offered. Each type has an hourly and a monthly cost. You will pay whichever cost is lower for your usage of this specific Server. Costs may differ between Locations.

Currency for all amounts is €. All prices exclude VAT.


# Get Instance

The instance of the `ServerTypesController` class can be accessed from the API Client.

```
serverTypesController := client.ServerTypesController()
```
